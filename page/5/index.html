灰度发布CanaryRelease

发表于 2019-02-27 | 更新于 2019-09-30 | 分类于 canary-release

灰度发布,也叫金丝雀发布,是指使服务平滑过渡的一种发布方式.在灰度发布之上可以进行A/B测试,如果用户对B没有什么反对意见,那么就逐步扩大范围(这个过程也叫灰度放量),把所有的用户都迁移到B上来.

灰度发布能及早获得用户反馈,完善产品功能,同时可以加强与用户的互动,降低产品所影响的用户范围.

核心概念

  1. 前提是新功能的引入.
  2. 可以应用在系统的某个部分上,比如UI,或者后端某个子服务.总之是系统流程中的某个环节.
  3. 一开始是少量用户介入,也可能用户没有感知.
  4. 灰度发布的核心是如何快速的切换,灰度发布不会减少错误,但是可以减少停机时间.

股市思考

发表于 2019-01-07 | 更新于 2019-10-01 | 分类于 stock

股市到底分析什么?

什么是价值?每人见解都不一样.如果大多数人都认可的价值,会被市场认可吗?基本面,很多时候是心理面.
其实大家觉得的价值,可以就近认为是价格.价格的因素之一是情绪,研究情绪是有意义的.

通过大数据发现,任何一支股票的后续走势只有三种:上涨,横盘,下跌.分析不能预测未来,但是可以应对未来的变化.

历史不会重演,但是会有相似.通过大量经验形成的预案,在应对未来的时候,大概率会帮助截断亏损,放大利润.
比如:预案会涨,但是没涨,不符合预期,可能会跌,就应该收手;如果涨了,符合预期,就应该持有.

预案到执行,还要过心理关.比如回撤的考验,止损的考验等等.市场喜欢冷血无情的交易者.

可转债

发表于 2019-01-06 | 更新于 2019-10-01 | 分类于 convertible-bond

可转换债券是债券持有人可按照发行时约定的价格将债券转换成公司的普通股票的债券.

转换价格 盈亏比
$$\dfrac{票面价格}{转股价格}\times正股价格=转换价格$$ $$\dfrac{可转债实时价格-转换价格}{转换价格}=盈亏比$$

盈亏比为负,适合买入,盈亏比为正,不适合买入.

  1. 正股一直跌,转债大概会跌.
  2. 下修转股价,转债大概会涨.

财报阅读

发表于 2019-01-04 | 更新于 2019-10-01 | 分类于 financial-report

财报阅读Tips

  1. 关注收入: 经营性收入(70%), 投资性收入, 筹资

  2. 关注几张表: 利润表, 现金流量表, 资产负债表

  3. 关注几个指标: 每股收益, 净资产收益, 总资产收益率, 主营业务收入增长率

    主营增长超过10%视为成长型; 5%-10%为稳定期; 小于5%视为衰退期

PE PB PEG

  1. PE: 市盈率, 预计多久回本

  2. PB: 市净率

  3. PEG

  4. 股息率：年度分红金额/股票价格

  5. 分红率：分红金额/净利润

  • 损失厌恶 - 对避免损失有一种强烈的偏好, 不挣钱远比赔钱重要
  • 沉淀成本 - 重视已经花掉的钱却无视将来可能要花掉的钱

  • 锚定效应 - 在做出一个涉及不确定的决策时过于依赖容易获得的信息
