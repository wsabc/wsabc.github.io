财报阅读

发表于 2019-04-20 | 更新于 2019-06-12 | 分类于 financial-report

财报阅读Tips

  1. 关注收入: 经营性收入(70%), 投资性收入, 筹资

  2. 关注几张表: 利润表, 现金流量表, 资产负债表

  3. 关注几个指标: 每股收益, 净资产收益, 总资产收益率, 主营业务收入增长率

    主营增长超过10%视为成长型; 5%-10%为稳定期; 小于5%视为衰退期

PE PB PEG

  1. PE: 市盈率, 预计多久回本

  2. PB: 市净率

  3. PEG

  4. 股息率：年度分红金额/股票价格

  5. 分红率：分红金额/净利润

可转债

发表于 2019-04-20 | 更新于 2019-06-12 | 分类于 convertible-bond

可转换债券是债券持有人可按照发行时约定的价格将债券转换成公司的普通股票的债券.

转换价格 盈亏比
$$\dfrac{票面价格}{转股价格}\times正股价格=转换价格$$ $$\dfrac{可转债实时价格-转换价格}{转换价格}=盈亏比$$

盈亏比为负,适合买入,盈亏比为正,不适合买入.

  1. 正股一直跌,转债大概会跌.
  2. 下修转股价,转债大概会涨.

Java IO

发表于 2019-03-23 | 更新于 2019-09-12 | 分类于 IO

Java IO 是一套Java用来读写数据(输入和输出)的API.大部分程序都要处理一些输入,并由输入产生一些输出.Java为此提供了java.io包.Java.io包主要涉及文件,网络数据流,内存缓冲等的输入输出.

Java IO基础

  1. 什么是流Stream?

流从概念上来说是一个连续的数据流.你既可以从流中读取数据,也可以往流中写数据.流有方向即输入和输出.

  1. 什么是字节流?字符流?

在Java IO中流既可以是字节流(以字节为单位进行读写,InputStream,OutputStream为基类),也可以是字符流(以字符为单位进行读写,Reader,Writer是基类).
