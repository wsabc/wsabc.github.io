海龟交易法则

发表于 2019-01-23 | 更新于 2019-03-11 | 分类于 stock | 阅读次数：

克服几个心理障碍

  • 损失厌恶 - 对避免损失有一种强烈的偏好, 不挣钱远比赔钱重要
  • 沉淀成本 - 重视已经花掉的钱却无视将来可能要花掉的钱
  • 处置效应 - 早早兑现利润, 却让损失持续下去
  • 锚定效应 - 在做出一个涉及不确定的决策时过于依赖容易获得的信息

类似的还有近期偏好, 结果偏好, 潮流效应, 信奉小数法则.

海龟培训

  • 破产风险 - 由于一连串的失败而赔光所有的钱
  • 期望值 - 找到一个期望值为正的法则并长期坚持

掌握优势, 管理风险, 坚定不移, 简单明了.

通过系统抓住每一个趋势, 大部分的利润都来源于两三次的成功交易, 大部分的交易都是失败的. 避免结果偏好, 相信正期望值的威力.

原版海龟交易法则

  1. 买什么? 选择流动性大的市场/股票
  2. 买多少? 结合具体市场的波动性来标准化头寸规模,使得不同市场绝对盈亏相同
  3. 什么时候买? 唐奇安通道突破系统
  4. 什么时候止损? 往相反方向波动2N即止损
  5. 什么时候止盈? 系统1采用10日突破原则, 系统2采用20日突破原则
  6. 怎么买卖? 严格遵守系统信号,保持一致性

波动性 N 计算

N = ATR(真是波动幅度均值) = 真实波动幅度的20日指数移动平均值, 代表一个市场在一天内的平均波动幅度, 包括开盘跳空的情况.

TR(真实波动幅度) = Max(H−L, H−PDC, PDC−L)

N = (19 ∗ PDN + TR) / 20 (首日的N = TR/20)

绝对波动幅度(货币) = N * 每一点所代表的货币 (如沪深300股指期货,一点代表300人民币)

头寸规模单位 = 账户净值1% / 绝对波动幅度, 计算出来的头寸一天内的波动就是账户的1%.

市场 头寸上限
单个市场 4
高度关联的多个市场 6
松散关联的多个市场 10
单个方向(多头或空头) 12

缩减账户规模: 按照初始账户规模 - 盈利或亏损的2倍

突破系统

  • 系统1: 20日突破为基础的短期系统
  • 系统2: 55日突破为基础的长期系统

突破不是突破均线, 而是突破过去一定时间内的最高点或者最低点, 而且不是以收盘价为准. 突破即时入市.

  • 盈利性突破: 根据10日退出法则获利(无论上一次是否发生交易), 则上一次属于盈利性突破, 此时要忽略当前突破.
  • 亏损性突破: 突破获利退出之前发生了2N幅度的不利变动. 新突破视为有效突破.
  • 55日突破将作为保障性突破点, 无论上一次是亏损突破还是盈利突破.

仓位管理

  • 符合预期按照0.5N的区间逐步建仓.
  • 初始仓位止损波动-2N, 如果逐步加仓, 止损价距离最近补仓价-2N; 如果有跳空的情况, 则之前的止损价不变, 新单位的头寸按照新的止损价.
  • 坚定不移的遵守入市规则和加减仓规则.
