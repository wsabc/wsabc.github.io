Mac环境变量

  1. 定义环境变量: JETTY_BASE=/tmp/base
  2. 设置环境变量: export JETTY_BASE
  3. 应用配置: source ~/.bash_profile
  4. 检查变量: echo $JETTY_BASE

终端export的变量,只在这个终端生效,是临时变量,如果需要长期保存,需要设置在.bash_profile之类的启动文件里.
