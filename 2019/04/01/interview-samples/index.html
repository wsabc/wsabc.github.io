面试题Sample

发表于 2019-04-01 | 更新于 2019-10-01 | 分类于 interview | 阅读次数：

网上看的一些面试题,记录下来.

  1. 索引的数据结构及原理, 哪些字段应当创建索引?
  2. 针对于一个慢 SQL 的优化思路.
  3. 数据库水平垂直拆分的方案, 需要了解业界常用的 MyCAT, sharding-sphere 等中间件.
  4. 了解<阿里巴巴 Java 开发手册>
  5. CAP 定理, 知道分布式系统会带来的一些问题以及各个应用权衡的方式.
  6. 对 Dubbo, SpringCloud 等分布式框架的使用, 最好是要了解原理.

  7. 分布式锁, 分布式事务, 分布式缓存, 分布式 ID, 消息中间件等.分布式中的负载算法:权重, Hash, 一致性 Hash, 故障转移, LRU

  8. 谈谈你做过项目中印象较深或自认为做的比较好的地方.

  9. AOP 谈谈它的实现原理. 以及不同实现的优缺点.
  10. JMM 内存模型, 如何划分的? 分别存储什么内容? 线程安全与否?
  11. 类加载机制, 谈到双亲委派模型后会问到哪些违反了双亲委派模型？为什么？为什么要双亲委派？好处是什么？
  12. 平时怎么使用多线程？有哪些好处？线程池的几个核心参数的意义？
  13. 线程间通信的方式？
  14. HashMap 的原理？当谈到线程不安全时自然引申出 ConcurrentHashMap, 它的实现原理？
  15. 分库分表如何设计？垂直拆分、水平拆分？
  16. 业务 ID 的生成规则，有哪些方式？
  17. SQL 调优？平时使用数据库有哪些注意点？

  18. 当一个应用启动缓慢如何优化？

  19. 谈谈你所理解的 HTTP 协议？

  20. 对 TCP 的理解？三次握手？滑动窗口？
  21. 基本算法，Base64 等。
  22. Java 内存模型，Happen Before 的理解。
  23. 对 Java 锁的理解？
  24. synchronize，Lock他们两者的区别以及优缺点呢？
  25. 分布式锁，
  26. 线程间通信的多种方式？
  27. 限流算法？单机限流？分布式限流？
  28. 提到了 Guava Cache ,了解它的实现原理嘛？
  29. 如何定位一个线上问题？
  30. CPU 高负载？OOM 排查等？
  31. 谈谈对 Netty 的理解？

  32. Netty 的线程模型？

  33. 服务化框架的选型和差异？

  34. 一起探讨了 SpringCloud、Dubbo、Thrift 的差异，优缺点等。
  35. 一致性 Hash 算法的原理？
  36. 谈谈你理解的 Zookeeper？
  37. 如何处理 MQ 重复消费？
  38. 客户端负载算法？轮询、随机、一致性 Hash、故障转移、LRU 等。。
  39. long 类型的赋值是否是原子的？
  40. volatile 关键字的原理及作用？happen Before？
  41. 谈谈对微服务的理解，好处以及弊端？
  42. 分布式缓存的设计？热点缓存？
  43. 给一个场景，谈谈你的架构方式。

  44. 在网页上点击一个按钮到服务器的整个流程，尽量完整。

  45. 从一个日志文件中根据关键字读取日志，记录出现的次数，最后按照次数排序打印。

  46. 写一个 LRU 缓存。
  47. 交替打印奇偶数
